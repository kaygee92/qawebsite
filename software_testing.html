Software Testing Tools

There are many different tools that trainees use in Quality Assurance. The trainee will be using Bugzilla and Selenium to learn about bug testing. The trainees learn about Software Development Life Cycle and practice the act of code & fix (cowboy) methodology throughout their time at Year Up.

The trainees use the book Software Testing: Second Edition by Ron Patton in the training room. The trainees learn about High and low level testing.
